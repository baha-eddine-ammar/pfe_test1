# Server Room Supervision - Internship Report Documentation

This folder contains report-ready documentation for the Server Room Supervision web application.

Files:

- `01-executive-summary.md`: general project presentation and objectives.
- `02-technologies-used.md`: technologies grouped by field.
- `03-complete-feature-list.md`: complete feature descriptions with files and technologies.
- `04-technical-architecture.md`: layered architecture explanation.
- `05-three-month-sprint-plan.md`: realistic 3-month internship sprint plan.
- `06-report-ready-sections.md`: ready-to-use report paragraphs.
- `07-screenshot-suggestions.md`: suggested screenshots for the internship report.

Security note:

The project uses private configuration values through `.env`, including service tokens and API credentials. These values must not be copied into the internship report, committed to source control, or shared publicly.
